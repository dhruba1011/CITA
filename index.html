Certificate in Information Technology Applications (CITA)

The CITA (Certificate in Information Technology Applications) course is a comprehensive training program designed to equip students with the foundational and intermediate skills necessary to thrive in the software and application development field.

Course Highlights

The CITA course includes fundamental Concepts od Computer, Application of Windows Operating System and the details of office package

1. Fundamental of Computer

  • Variables, data types, and operators
  • Control structures (if, switch, loops)
  • Functions and arrays
  • Pointers and structures
  • File handling in C

2. Windows - Operating System

  • HTML structure and tags
  • Forms, tables, and lists
  • Embedding images, videos, and links
  • Semantic HTML5 elements

3. MS Word

  • Java syntax and structure
  • OOP principles (classes, objects, inheritance, polymorphism)
  • Exception handling and file I/O
  • GUI development with Swing

4. MS Excel

  • Programming
  • Form Designing
  • Data Base Connectivity

Why Choose CITA?

  • Balanced mix of coding, web development, and database skills
  • Ideal for students preparing for entry-level jobs or further studies in IT
  • Hands-on projects and assignments
  • Enhances both technical knowledge and logical reasoning
