Certificate in Information Technology Applications (CITA)

The CITA (Certificate in Information Technology Applications) course is a comprehensive training program designed to equip students with the foundational and intermediate skills necessary to thrive in the software and application development field.

Course Highlights

The CITA course includes fundamental Concepts od Computer, Application of Windows Operating System and the details of office package

1. Fundamentals of Computer

  • A computer is an electronic device that processes data and performs tasks based on instructions (programs).
  • Basic components: CPU (Processor), RAM (Memory), Storage (HDD/SSD), Input/Output devices.
  • Works on the IPO cycle (Input → Process → Output).
  • Uses binary system (0s and 1s) for data representation.
  • Types: Analog, Digital, Hybrid, Supercomputers, Mainframes, PCs.
  • Essential software includes Operating System (OS), Applications, and Utilities.

2. Windows - Operating System

  • Windows is a graphical OS developed by Microsoft, widely used in PCs.
  • Provides a user-friendly GUI (Graphical User Interface) with icons, menus, and windows.
  • Manages hardware, software, memory, and file systems.
  • Supports multitasking (running multiple programs simultaneously).
  • Common versions: Windows 7, 8, 10, 11 (Windows 7 was popular for its stability).
  • Features include Start Menu, Taskbar, File Explorer, Control Panel, and Security tools.

3. MS Word

  • A word processing software for creating, editing, and formatting documents.
  • Used for letters, reports, resumes, and other text-based projects.
  • Key features: Font styling, Tables, Headers/Footers, Spell Check, Mail Merge.
  • Supports images, charts, and hyperlinks for enhanced documents.
  • File formats: .doc, .docx, .pdf (exportable).
  • Part of Microsoft Office Suite (along with Excel, PowerPoint, etc.).

4. MS Excel

  • A spreadsheet program used for data organization, calculations, and analysis.
  • Works with rows, columns, and cells (e.g., A1, B2).
  • Features: Formulas (SUM, AVERAGE), Functions (VLOOKUP, IF), PivotTables, Charts.
  • Used for budgeting, financial analysis, inventory tracking.
  • Supports macros (automation using VBA - Visual Basic for Applications).
  • File formats: .xls, .xlsx, .csv.

5. MS PowerPoint

  • A presentation software for creating slideshows with text, images, and animations.
  • Used in business meetings, education, and conferences.
  • Features: Slide transitions, Animations, Templates, Speaker Notes.
  • Supports embedding videos, charts, and hyperlinks.
  • Can be exported to PDF, video, or image formats.
  • Part of Microsoft Office Suite.

6. Visual FoxPro

  • A database management system (DBMS) and programming language by Microsoft.
  • Used for developing desktop database applications.
  • Supports SQL queries, forms, and reports for data handling.
  • Features: Fast data processing, OOP (Object-Oriented Programming) support.
  • Discontinued by Microsoft in 2015 but still used in legacy systems.
  • File extensions: .dbc (database), .prg (program files).
